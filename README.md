**Mail Hunter** is a powerful Chrome Extension designed to automate the process of finding targeted email leads from Google Search results. It filters data based on specific keywords, locations, and social media platforms (LinkedIn, Facebook, Instagram, etc.), exporting the results directly to CSV.

## üöÄ Features

* **Targeted Search:** Find emails based on Keywords (e.g., "Doctor", "CEO") and Location.
* **Platform Filtering:** Target specific networks like LinkedIn, Facebook, Twitter, or Instagram.
* **Domain Specific:** Filter by email providers (Gmail, Yahoo, Hotmail, Outlook).
* **Automated Scraping:** Automatically navigates through Google Search pages to collect data.
* **CSV Export:** Download a clean, organized `.csv` file with Email, Domain, Keyword, Location, and Platform.
* **Modern UI:** Clean, professional interface for a smooth experience.

## üìñ How to Use

1.  Click the **Mail Hunter** extension icon.
2.  **Keyword:** Enter your target niche (e.g., *Real Estate Agent*).
3.  **Location:** Enter the target country or city (e.g., *New York*).
4.  **Platform:** Select the social source (e.g., *LinkedIn*).
5.  **Email Domain:** Choose the email provider you want to target.
6.  Click **Start**. The tool will open a background tab and start collecting emails.
7.  Watch the "Collected Results" counter grow.
8.  Click **Stop** when you have enough leads.
9.  Hit **Download CSV** to save your leads!
